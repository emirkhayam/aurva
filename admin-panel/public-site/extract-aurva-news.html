Инструмент для извлечения новостей с сайта AURVA

Инструкция по использованию:

  1. Нажмите кнопку "Извлечь новости"
  2. Дождитесь завершения загрузки (10-15 секунд)
  3. Скопируйте JSON из текстового поля ниже
  4. Сохраните в файл aurva-news-data.json
  5. Передайте файл Claude для импорта в базу данных
