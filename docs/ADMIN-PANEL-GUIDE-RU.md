## Вход в систему

### Доступ к админ-панели

Админ-панель доступна по адресу: `https://aurva.kg/admin` (или `http://localhost:5000/admin` для разработки)

### Процесс входа

1. Откройте страницу входа
2. Введите ваш email (например: `admin@aurva.kg`)
3. Введите пароль
4. Нажмите кнопку **"Войти"**

**Важно:**
- После 5 неудачных попыток входа аккаунт будет заблокирован на 15 минут
- Токен доступа действителен 24 часа
- При закрытии браузера вам потребуется войти снова

### Первый вход

При первом входе рекомендуется:
1. Перейти в раздел **"Профиль"**
2. Изменить пароль по умолчанию
3. Убедиться, что ваше имя и email указаны верно

Используйте боковое меню для перехода к различным разделам системы.

## Управление новостями

### Просмотр всех новостей

1. Откройте раздел **"Новости"** в боковом меню
2. Вы увидите список всех новостей с информацией:
   - Заголовок
   - Категория
   - Дата публикации
   - Статус (опубликовано / черновик)

### Создание новости

1. Нажмите кнопку **"+ Добавить новость"**
2. Заполните обязательные поля:
   - **Заголовок**: Название статьи (макс. 500 символов)
   - **Краткое описание**: Краткий текст для превью (100-200 символов)
   - **Содержание**: Полный текст статьи (используйте Rich Text редактор)
   - **Категория**: Выберите одну из категорий:
     - Регулирование
     - События
     - Аналитика
     - Другое

3. Дополнительные параметры:
   - **Изображения**: Загрузите до 10 изображений
     - Поддерживаемые форматы: JPEG, PNG, WebP
     - Макс. размер: 5 МБ на файл
     - Изображения автоматически оптимизируются
   - **URL изображений**: Можно указать ссылки на внешние изображения (через запятую)

4. Установите переключатель **"Опубликовать"**, чтобы статья стала видна на сайте

5. Нажмите **"Создать"**

### Загрузка изображений - Новое!

**Превью перед загрузкой:**
- После выбора файлов вы сразу увидите миниатюры
- Можно удалить ненужные изображения до загрузки
- Отображается размер каждого файла

**Автоматическая оптимизация:**
- Все изображения автоматически сжимаются
- Конвертируются в формат WebP для быстрой загрузки
- Максимальный размер: 1920x1920 пикселей
- Качество: 85% (оптимальный баланс)

### Редактирование новости

1. Нажмите на новость в списке или кнопку "Редактировать"
2. Измените необходимые поля
3. Нажмите **"Сохранить"**

**Совет**: URL статьи (slug) генерируется автоматически из заголовка и не меняется при редактировании

### Удаление новости

1. Нажмите кнопку **"Удалить"** рядом с новостью
2. Подтвердите удаление

**Внимание**: Удаленные новости нельзя восстановить!

## Управление контактами

Раздел для работы с заявками, поступившими через контактную форму на сайте.

### Просмотр контактов

1. Откройте раздел **"Контакты"**
2. Вы увидите список всех заявок с информацией:
   - Имя / Название компании
   - Телефон
   - Email (если указан)
   - Статус обработки
   - Дата подачи заявки

### Статусы контактов

- **Новый** (new) - заявка только что поступила
- **Связались** (contacted) - вы связались с клиентом
- **Обработано** (processed) - заявка полностью обработана
- **Отклонено** (rejected) - заявка отклонена

### Изменение статуса

1. Найдите нужную заявку
2. Выберите новый статус из выпадающего списка
3. Статус обновится автоматически

### Добавление заметок

1. Откройте детали контакта
2. В поле **"Заметки"** введите комментарии (например, результат звонка)
3. Нажмите **"Сохранить"**

### Конвертация в участника - Новое!

Если контакт хочет стать участником ассоциации:

1. Откройте детали контакта
2. Нажмите кнопку **"Конвертировать в участника"**
3. Форма создания участника откроется с предзаполненными данными
4. Дополните информацию (описание, логотип, сайт)
5. Сохраните

**Преимущество**: Не нужно вручную переносить данные!

### Массовые операции - Новое!

Для обработки нескольких контактов одновременно:

1. Отметьте чекбоксы рядом с нужными контактами
2. Выберите действие:
   - Изменить статус на "Связались"
   - Изменить статус на "Обработано"
   - Удалить выбранные
3. Подтвердите операцию

## Управление участниками

Управление компаниями и организациями - участниками AURVA.

### Создание участника

1. Откройте раздел **"Участники"**
2. Нажмите **"+ Добавить участника"**
3. Заполните поля:
   - **Название**: Полное название компании (обязательно)
   - **Описание**: Краткое описание деятельности
   - **Веб-сайт**: URL сайта компании (формат: https://example.com)
   - **Логотип**: Загрузите изображение логотипа
     - Рекомендуемый размер: 400x400 пикселей
     - Формат: PNG с прозрачностью
   - **Дата вступления**: Когда компания стала участником
   - **Порядок отображения**: Число для сортировки (меньше = выше)
   - **Активен**: Включите, чтобы участник отображался на сайте

4. Нажмите **"Создать"**

### Редактирование / Удаление

Аналогично разделу новостей - нажмите соответствующую кнопку в списке.

## Управление партнёрами

Управление партнёрами ассоциации (не путать с участниками!).

### Отличие партнёров от участников

- **Участники** - компании, которые являются членами AURVA
- **Партнёры** - внешние организации, с которыми AURVA сотрудничает

### Создание партнёра

Процесс аналогичен созданию участника, но с дополнительными полями:

- **Заголовок модального окна**: Полное название для всплывающего окна
- **Описание для модалки**: Подробная информация о партнёре
- **Преимущества**: Список преимуществ сотрудничества

## Управление командой

Управление информацией о сотрудниках и руководстве AURVA.

### Категории

- **Руководство** (leadership) - президент, директора
- **Совет** (council) - члены совета
- **Другое** (other) - остальные сотрудники

### Добавление члена команды

1. Откройте раздел **"Команда"**
2. Нажмите **"+ Добавить члена команды"**
3. Заполните:
   - **Имя**: ФИО
   - **Должность**: Официальная должность
   - **Фото**: Профессиональное фото (рекомендуется 600x600)
   - **Биография**: Краткая информация о человеке
   - **Категория**: Выберите из списка
   - **Порядок отображения**: Для сортировки

## Решение проблем

### "401 Unauthorized" - Сессия истекла

**Причина**: Токен доступа действует 24 часа

**Решение**:
- Система автоматически попытается обновить токен
- Если не получилось - войдите в систему заново

### "Файл слишком большой"

**Причина**: Файл превышает 5 МБ

**Решение**:
- Используйте онлайн-сервис для сжатия изображения
- Или измените размер изображения перед загрузкой
- Система автоматически оптимизирует изображения, но они должны быть < 5 МБ

### "Недостаточно прав"

**Причина**: Ваша роль не позволяет выполнить действие

**Решение**:
- Убедитесь, что ваша роль - "admin" (не "moderator")
- Свяжитесь с главным администратором для повышения прав

### Изображения не загружаются на сайт

**Возможные причины**:
1. Новость не опубликована (переключатель "Опубликовать" выключен)
2. Неправильный формат URL изображения
3. Проблемы с правами доступа к папке uploads/

**Решение**:
1. Убедитесь, что новость опубликована
2. Проверьте URL изображений (должны начинаться с http:// или https://)
3. Обратитесь к техническому специалисту

### Браузер не открывает админ-панель

**Решение**:
- Очистите кэш браузера (Ctrl+Shift+Delete)
- Попробуйте режим инкогнито
- Убедитесь, что backend сервер запущен (для разработки)

## Советы по эффективной работе

### Для новостей
1. Пишите краткие, информативные заголовки
2. Указывайте правильную категорию - это помогает пользователям фильтровать
3. Используйте качественные изображения (они оптимизируются автоматически)
4. Сохраняйте черновики, если статья не готова к публикации

### Для контактов
1. Регулярно проверяйте новые заявки
2. Добавляйте заметки после каждого контакта
3. Используйте массовые операции для обработки групп контактов
4. Конвертируйте потенциальных участников сразу

### Для участников/партнёров
1. Используйте логотипы в формате PNG с прозрачностью
2. Заполняйте описания - это важно для SEO
3. Указывайте корректные ссылки на веб-сайты
4. Используйте "Порядок отображения" для важных участников
