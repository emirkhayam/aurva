## 🚀 Как Запустить

### 1. Локальная Разработка

```bash
# 1. Создать .env
cp .env.example .env
nano .env  # Заполнить переменные

# 2. Запустить все сервисы
docker-compose -f docker-compose.dev.yml up --build

# 3. Открыть браузер
# 🌐 Публичный сайт:  http://localhost
# 🔐 Админка:         http://localhost/admin
# 🔌 API:             http://localhost/api
```

### 2. Production Деплой

```bash
# На сервере (Coolify делает автоматически)
docker-compose up --build -d
```

## 📝 Workflow Разработки

### Редактирование Frontend

```bash
# 1. Редактируйте ТОЛЬКО в frontend/
nano frontend/index.html

# 2. Синхронизируйте для деплоя
./sync-frontend.sh

# 3. Перезагрузите nginx
docker-compose -f docker-compose.dev.yml restart nginx
```

### Редактирование Admin Panel

```bash
# Просто редактируйте - hot reload работает
nano admin-panel/src/App.tsx
# Изменения появятся автоматически на http://localhost/admin
```

### Редактирование Backend

```bash
# Просто редактируйте - nodemon перезагрузит
nano backend/src/routes/newsRoutes.ts
# API обновится автоматически
```
